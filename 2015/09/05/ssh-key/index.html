ssh key manage

有时我们需要同时在github, gitlab等多个代码托管平台工作又不想使用相同的ssh key，那么就需要根据不同的host来配置。

create key: id_rsa_moz

在输入文件名称时注意与已有的区分

添加到ssh agent让ssh识别新的私钥

配置~/.ssh/config文件

假如是对同一主机设置的不同key的话要把配置中Host（别名）区分出来，连接时通过不同的host来使用不同的key。
